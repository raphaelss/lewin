NAME

lewin — a compositional calculator

DESCRIPTION

The Lewin calculator is a tool for operations with pitch class sets and related concepts that is useful both for musical analysis and composition.

CONTRIBUTING

We are currently refactoring the code to be able to cleanly introduce new features. There is a mirror of the repository at github.com/raphaelss/lewin and you can open issues there for bug reports and feature requests or send patches via pull requests. You are also welcome to send bug reports, feature suggestions and patches via email to one of the authors.
